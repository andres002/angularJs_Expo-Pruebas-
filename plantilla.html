API( Application Programming Interface.) Interfaz de programación de aplicaciones.

Se definen como un conjunto de funciones globales para realizar tareas comunes tales como:
  • comparar objetos
  • iterar objetos
  • convertir datos

Nosotros podemos tener acceso a estas funciones mediante el objeto "Angular" y algunas de las mas conocidas son:

API Descripcion
angular.lowercase() convierte una String a minuscula
angular.uppercase() convierte una String a Mayuscula
angular.isString() retorna true si la referencia es una String
angular.isNumber() retorna true si la referencia es un numero

Angular animation

AngularJS nos ofrece trancisiones animadas con ayuda de CSS

¿Qué es una animación?

Es cuando una transformación de HTML nos dá la ilusión de movimiento


¿Qué necesitas?

Necesitas incluir la libreria Animate de AngularJs

<script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-animate.js"></script>

luego necesitas referenciar al modulo a animar en tu aplicacion

<body ng-app="ngAnimate">

Pero si ya tienes el nombre de una aplicación añade ngAnimate como una dependencia en tu modulo de la aplicacion

var app = angular.module('myApp', ['ngAnimate']);

Entonces... ¿Qué hace ng-Animate?

ng-Animate no anima los elementos de tu Html
